## Главный вывод

Техническая база сайта выше среднего по нише: 31 товарная страница — полноценная статика с Product+AggregateOffer JSON-LD, FAQPage на инфостраницах, canonical, sitemap, быстрый RU-хостинг. Но **бренд KOMUI не существует за пределами собственного домена**: ноль упоминаний в поиске, ноль отзывов, ноль соцсетей, имя перехвачено персонажем аниме, Ozon-магазин называется «GETO» и не связан с брендом публично. AI-системы не могут рекомендовать магазин, о существовании которого не знают. Второй якорь — контент: описания товаров скопированы с Ozon (каннибализация более авторитетным доменом) с keyword-stuffing-хвостами.

## Дорожная карта

### 🔴 Критично (неделя 1: nginx + быстрые файлы)

1. **Починить soft-404.** Любой несуществующий URL отдаёт 200+index.html; у каждого товара есть дубль со слэшем. Сайт НЕ использует path-роутинг (только хэши #catalog/#cart), поэтому fallback можно убрать безопасно:
   `try_files $uri $uri.html =404;` + `error_page 404 /404.html` + `rewrite ^/(.+)/$ /$1 permanent;`
2. **301 www → apex.** www.komui.ru сейчас полный дубль сайта с кодом 200.
3. **Задеплоить llms.txt** (готовый текст — в отчёте AI-visibility) как статический файл с Content-Type: text/plain.
4. **Яндекс.Вебмастер + Google Search Console + Bing WMT**: верификация, sitemap, диагностика индексации — сейчас нет данных из первых рук.

### 🟠 Высокий приоритет (недели 2–4)

5. **Перенести все фото товаров с ir.ozone.ru на komui.ru** (в build-products.js) + WebP, width/height, srcset. Хотлинк с CDN Ozon — единая точка отказа для страниц, schema и картиночной выдачи.
6. **Яндекс-экосистема: Яндекс.Бизнес + YML-фид** (генерация из storefront-products.js) для Яндекс Товаров — прямой канал в выдачу Нейро/Алисы.
7. **Переписать 31 описание товара** уникально: убрать Ozon-дубли, «🔹»-структуру и спам-хвосты «футболка 2025, одежда для отаку…»; добавить факты (техника нанесения, посадка, замеры). Устранить противоречие 200 vs 240 г/м².
8. **Отзывы**: начать сбор на сайте (и на Ozon), пока рейтинга нет — показывать Ozon-рейтинг текстом со ссылкой. aggregateRating в schema — только после появления собственных отзывов.
9. **Строить сущность бренда**: сообщество VK + Telegram-канал (добавить в футер и sameAs), связать GETO↔KOMUI на Ozon, элемент Wikidata «KOMUI — бренд одежды», карточка Яндекс.Бизнес.
10. **Обогатить Product-schema**: size у каждого Offer, priceValidUntil, OfferShippingDetails, MerchantReturnPolicy (готовые шаблоны — в schema-отчёте). Удалить сломанный SearchAction (target с #catalog не работает).

### 🟡 Средний приоритет (месяц 2)

11. **Персонажные лендинги** по шаблону collections: Акацуки (6 товаров), Итачи (6+), Годжо (7), Сукуна — закрывают высокочастотники «футболка акацуки», «футболка годжо» без посадочных.
12. **Категорийные лендинги** «Аниме-футболки» (28 SKU) и «Аниме-худи» — сейчас категории живут только в JS-фильтре.
13. **Дедуплицировать SEO-тексты коллекций** (боилерплейт-абзацы идентичны), убрать мета-фразы «если вы пришли по запросу», починить «16 товар(ов)»; добавить H3-вопросы + FAQ-блоки с schema.
14. **Страница «О бренде KOMUI»**: производство в Дагестане, фото вышивки, основатель — главный E-E-A-T-актив, который не скопирует ни один конкурент. Брендовая почта info@komui.ru вместо smmshit@ya.ru.
15. **Даты** datePublished/dateModified + видимое «Обновлено» на всех страницах; честный lastmod в sitemap (сейчас у всех 47 URL дата билда).
16. **IndexNow** (ключ-файл + ping из build-скрипта) — кормит Bing и Яндекс одним внедрением.
17. **Производительность**: Cache-Control для статики, HSTS, brotli, defer для data/*.js, self-host Google Fonts (в РФ googleapis нестабилен), fetchpriority=high на LCP-изображение.
18. **Блог, первые 4 статьи**: «Как ухаживать за DTF-принтом», «Вышивка или принт: что выбрать» (уникальная экспертиза — обе техники в ассортименте), «Подарок фанату Наруто», «Что такое футболка-варёнка».

### 🟢 Низкий приоритет (месяц 3+)

19. Починить BreadcrumbList (реальные URL коллекций вместо #catalog/#strip), унифицировать @context, Organization → OnlineStore с contactPoint/address/taxID и растровым логотипом 600×600.
20. Обогатить delivery.html конкретными цифрами (стоимость от N ₽, сроки 2–7 дней) — сейчас лучший citability-контент сайта, но без цифр.
21. Посевы в аниме-сообществах (VK, Telegram, Pikabu), первые отзывы на irecommend/otzovik — питание Perplexity и Нейро.
22. Абзац-определение бренда под hero на главной («KOMUI — российский бренд аниме-мерча…») — канонический сниппет для LLM.
23. Проверить доступность komui.ru с зарубежных IP (боты OpenAI/Perplexity ходят из-за рубежа; часть RU-хостеров режет иностранные подсети).
24. Alt-тексты: починить генератор («с вышивкаом»), Content-Signal в robots.txt по желанию.

## Ожидаемый эффект

- Недели 1–4 (пп. 1–10): устранение всех индексационных блокеров + вход в Яндекс-экосистему → рост с 44 до ~60.
- Месяц 2 (пп. 11–18): контентное покрытие коммерческих интентов + первые сигналы сущности → ~70.
- Месяц 3+ (пп. 19–24): накопление отзывов и внешних упоминаний — самый медленный, но самый весомый фактор (Brand Authority сейчас 8/100 при весе 20%).

Полные отчёты пяти направлений (с nginx-сниппетами, готовым llms.txt и JSON-LD шаблонами) — в транскриптах субагентов этой сессии.
